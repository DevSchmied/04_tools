# 04_tools

This repository contains exercises and practical examples created to study and
practice developer tools commonly used in development.

The focus is on understanding how these tools work internally, how they are used
in real applications, and how they support performance, scalability, and reliability.

---

## Current Content

### Redis Exercises

The Redis section includes hands-on exercises and examples covering:

- Core data structures:
  - strings, lists, sets, sorted sets, hashes
- Caching concepts and TTL
- Pub/Sub messaging
- Streams and stream-based processing
- Transactions
- Rate limiting
- Lua scripting
- Session storage patterns
- Task queue concepts
- Manual sharding concepts
- Redis security and replication (theoretical and practical exercises)

The exercises include both:
- Go-based implementations
- Command-level practice and summaries

---

## Purpose

The goal of this repository is to:
- gain hands-on experience with backend development tools
- understand internal concepts beyond basic usage
- practice integration of tools into backend systems
- complement language- and project-focused learning
